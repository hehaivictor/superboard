# George Soros source index

    ## 使用边界

    - 本目录按女娲流程保存公开来源索引、研究摘要和董事会适配结果。
    - 不复制书籍、文章、访谈或外部 skill 正文；只保存链接、来源类型和自有提炼。
    - 对无法直接覆盖的问题，必须降低置信度并在 Super Board 输出中标注“待验证”。
